## Deployment naar Vercel

### Via Claude Code / Vercel CLI:

```bash
cd crossfit-020-uitverkoop
vercel
```

### Via Vercel Dashboard:

1. Ga naar [vercel.com](https://vercel.com)
2. Klik "Add New Project"
3. Upload deze folder
4. Deploy!

## Google Sheets Setup

De Google Apps Script is al geconfigureerd. Als je deze wilt aanpassen:

1. Open je Google Sheet
2. Ga naar: Extensies ‚Üí Apps Script
3. Pas het script aan indien nodig
4. Implementeer opnieuw als je wijzigingen maakt

## Bestanden

- `index.html` - De volledige website (single page)
- `google-apps-script.js` - Script voor Google Sheets (alleen voor referentie)
- `package.json` - Project configuratie
